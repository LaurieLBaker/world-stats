# World Stats: International and Ethical Statistics Learning Activities

Welcome to *World Stats*, a collection of open educational resources (OER) designed to teach statistics through real-world international datasets and socially meaningful questions.
This repo is the Github home of the learning activities for the Pressbooks World Stats handbook which contains reflection and coding activities in statistics.

## Goals

This repository aims to provide learning activities that:

- 🌍 Introduce students to datasets from countries around the world.
- 📊 Teach core statistical concepts using real case studies.
- 📖 Provide historical and social context alongside statistical analysis.

## Current Activities

### New Zealand: Māori Jury Representation and Simpson's Paradox

Explore how different ways of summarizing data can lead to different conclusions about representation. Students use jury pool data from New Zealand to investigate contingency tables, percentages, expected counts, and **Simpson's paradox**, while learning about the historical and legal context of Māori representation on juries.
